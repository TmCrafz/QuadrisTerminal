# TetrisTerminal
This is a small Tetris clone which runs completely in the terminal and only use ASCII chars. At the moment it only supports unix like systems. Tested only on Ubuntu.

## Controls

### Menu
- [W] Move Up
- [S] Move Down
- [Retrun] Select

### Game
- [A] Move Left
- [D] Move Right
- [O] Rotate Left
- [P] Rotate Right
- [0] Pause Game
